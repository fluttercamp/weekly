插件&App

  1. flutter_spinkit

    一系列 loading 指示动画

  2. annotation_route

    一个以注解方式实现的路由映射解决方案，基于 source_ge

  3. YHFlutterAdapter

    三行代码组件化集成 Flutter！可用于已有 iOS 项目，对原工程无侵入，无需更改原项目配置，集成后可直接以组件形式开发 Flutter 业务

  4. flutter-draggable-scrollbar

    可拖动的滚动条，用于在垂直列表中快速导航。附加选项是显示标签旁边的滚动拇指与有关当前项目的信息

  5. Flutter Permission handler Plugin

    权限插件，这个插件提供了一个跨平台(iOS, Android) API来请求和检查权限

  6. GradientBottomNavigationBar

    渐变背景色的底部导航按钮条

  7. Flutter Timeline Widget

    时间轴控件

  8. OpenNGO

    一个自己设计 UI 界面并用 Flutter 实现的 Demo 项目

  9. Wave Progress Bars

    波浪进度条控件

  10. json2dart-converter

    一个可以从原始 json 生成 dart 类的插

  11. giffy_dialog

    一个漂亮的自定义警告提示对话框

  12. Frideos

    一系列用于简化 Steam 和 BLoc 的使用和各种 Flutter 小部件(动画、转换、计时小部件、滚动文本等)的库

  13. UnicornDialer

    可以轻松创建自己的浮动操作按钮列表的库
